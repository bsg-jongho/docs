BSG Partners는 AWS 환경에 특화된 보안 컴플라이언스 자동화 솔루션을 제공합니다. 보안 담당자와 IT 감사인이 반복적인 수동 점검에서 벗어나, 데이터 기반으로 컴플라이언스를 관리할 수 있도록 지원합니다.

모든 제품은 **AWS Marketplace**를 통해 간편하게 구독하고 배포할 수 있습니다.

## 제품

<Card title="ISMS-P Intelligence Dashboard" icon="shield-halved" href="/isms-p-dashboard">
  AWS 환경을 매일 자동 스캔하여 546개 ISMS-P 보안 점검 항목의 준수 현황을 시각화합니다. 최대 20개 AWS 계정을 동시에 모니터링하고, 9개 대시보드 시트에서 접근통제부터 취약점 관리까지 한눈에 파악할 수 있습니다.
</Card>

## 왜 BSG Partners인가?

  <Card title="컴플라이언스 자동화" icon="robot">

  </Card>

  <Card title="비용 효율성" icon="coins">

  </Card>
