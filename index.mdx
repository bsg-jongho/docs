BSG Partners는 2000년 설립 이래 SAP/ERP 클라우드 마이그레이션, Cloud Native, Managed Service, DevOps/MLOps/LLMOps 분야에서 엔터프라이즈 IT 서비스를 제공해 온 기업입니다. 2016년 국내 최초(세계 19번째)로 AWS SAP Competency Partner를 달성했으며, 현재 AWS Marketplace를 통해 다양한 산업 분야의 클라우드 기반 솔루션 및 컨설팅을 제공하고 있습니다.

이 사이트는 BSG Partners가 AWS Marketplace에 등록한 제품의 기술 문서를 제공합니다.

## 제품 문서

<Card title="ISMS-P Intelligence Dashboard" icon="shield-halved" href="/isms-p-dashboard">
  AWS 환경에서 ISMS-P 인증 기준 546개 항목을 매일 자동 점검하고, 최대 20개 계정의 보안 컴플라이언스 현황을 하나의 대시보드에서 통합 관리할 수 있는 솔루션입니다.
</Card>

전체 제품 목록은 [BSG 등록 제품 소개](https://marketplace.bsggo.com/)에서 확인할 수 있습니다.
