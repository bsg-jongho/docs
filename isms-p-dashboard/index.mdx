<Info>
  ISMS-P 인증 심사를 준비하거나 보안 컴플라이언스를 지속적으로 관리해야 하는 보안팀을 위한 솔루션입니다. 매일 새벽 02:00(KST)에 AWS 환경을 자동 스캔하여, 수동 점검 없이도 현재 보안 상태를 정확하게 파악할 수 있습니다.
</Info>

## 주요 특징

- **546개 보안 점검 항목**을 매일 자동으로 실행하여 ISMS-P 인증 기준 전체를 커버합니다.
- **최대 20개 AWS 계정**을 동시에 모니터링하여 멀티 계정 환경에서도 통합 관리가 가능합니다.
- 컴플라이언스 스캔 결과, Security Hub 발견사항(GuardDuty/Inspector), CloudTrail 보안 이벤트를 **하나의 대시보드에서 통합** 분석합니다.
- 서버리스 아키텍처 기반으로 별도 인프라 운영 없이 **AWS Marketplace 구독만으로 바로 사용**할 수 있습니다.
